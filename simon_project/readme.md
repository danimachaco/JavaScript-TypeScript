Simón Dice – Versión 1
Proyecto en JavaScript para recrear una versión por consola del clásico juego Simón Dice, donde el jugador debe memorizar y reproducir secuencias de colores generadas aleatoriamente.
📌 Objetivo del proyecto
Este proyecto busca practicar:
Manejo de arrays y control de índices.
Uso de enumerados simulados mediante objetos.
Validación de entradas del usuario.
Control del flujo mediante funciones bien definidas.
Lectura asíncrona con readline en Node.js.
🎮 Cómo funciona el juego
El programa da la bienvenida y pide el nombre del jugador.
Se genera una secuencia completa de colores al inicio.
La primera ronda muestra 3 colores.
Si el jugador acierta, la secuencia mostrada aumenta en 1 color.
El jugador introduce los colores usando las letras:
R = Rojo
V = Verde
A = Azul
D = Dorado
El juego termina cuando:
El jugador falla, o
Llega al máximo de colores permitido.
